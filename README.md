# 🌾 Crop Trading Platform

A web application that connects farmers and salesmen for direct crop trading.

## 🚀 Features
- 👨‍🌾 Farmer dashboard (add crops)
- 🧑‍💼 Salesman marketplace (buy crops)
- 📦 Order management system
- ✅ Approve / Reject orders
- 📞 Contact sharing after approval

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript, Boostrap
- Backend: Flask (Python)
- Database: MySQL

## 📌 Future Improvements
- Payment integration
- Real-time notifications
- Mobile-friendly UI
